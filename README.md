## 📜 진행 방식
💡 실습은 본인의 레포지토리에서 진행하고, 해당레포에서는 학습 내용을 정리하고 공유합니다.
-  매주 정리한 자료를 PR 로 올립니다.
-  매주 자신이 정리한 내용을 10분 내로 발표합니다.
-  매주 해당 주차의 Issue를 함께 리뷰 및 논의합니다.
-  PR 및 이슈의 제목은 아래와 같이 합니다.
```
[X주차]_SectionX_제목_이름
```
